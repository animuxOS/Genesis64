Property Details

The "dirty" property

  "dirty"                    gboolean              : Read

True if the layouts have changed and need to be saved to a file.

Default value: FALSE

The "master" property

  "master"                   GdlDockMaster         : Read / Write

GdlDockMaster object which the layout object is attached to.
